What is the Pitt-Google Alerts Broker?

The Pitt-Google Broker (PGB) is a cloud-based, alert distribution service designed to provide near real-time processing of alerts from the Large Synoptic Survey Telescope (LSST) and Zwicky Transient Facility (ZTF). In addition to providing several value added data products (see below), the PGB provides public access to all ingested alerts from these surveys.

The PGB is designed to run on the Google Cloud Platform (GCP), and supports the deployment of downstream brokers built by external development teams. This allows users to deploy small scale projects for free and large projects on a pay-as-you-need basis. For an overview on available data products and how to access them in the cloud, see theGetting Started guides.
