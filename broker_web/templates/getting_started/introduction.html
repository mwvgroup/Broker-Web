Getting Started

The Pitt-Google Broker (PGB) is designed to provide a value-added copy of astronomical alerts in a way that supports the development and deployment of astronomical analysis pipelines. It is built from the ground up to provide a modular and scalable service that users can use to support a variety of parallel and down stream services. The Getting Started guide provides an overview for each of these data products and services, including examples of how to deploy your own alert based tools.
